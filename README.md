## О проекте

Бот ведёт партию в стиле классической городской мафии прямо в Telegram-группе:

- **Ночь** — чат молчит (удаление сообщений + мут), роли ходят в ЛС
- **День** — обсуждение ночи; мёртвые молчат
- **Голосование** → подтверждение казни → снова ночь
- Анимации заката / рассвета (`sunset.mp4`, `sunrise.mp4`)
- Независимые языки: **русский** в личке и **украинский** в чате (и наоборот)

## Быстрый старт

### 1. Бот в BotFather

1. Создайте бота у [@BotFather](https://t.me/BotFather) → получите токен  
2. `/setprivacy` → **Disable** (иначе бот не видит сообщения в группе)

### 2. Установка

```bash
git clone https://github.com/YOUR_USER/YOUR_REPO.git
cd YOUR_REPO

python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux / macOS
source .venv/bin/activate

pip install -r requirements.txt
cp .env.example .env   # Windows: copy .env.example .env
```

Откройте `.env` и укажите токен:

```env
BOT_TOKEN=123456:ABC-DEF_your_telegram_bot_token
```

### 3. Запуск

```bash
python main.py
```

На Windows можно просто запустить `start.bat`.

## Как играть

1. Напишите боту в личке `/start` → «Добавить бота в чат»
2. В группе сделайте бота **админом** с правами:
   - удаление сообщений  
   - ограничение участников  
3. Каждый игрок один раз открывает бота в ЛС (`/start`)
4. В группе: `/game` → «Присоединиться» → хост жмёт «Начать игру»

```
/game  → набор
  ↓
 ночь  → действия в ЛС
  ↓
 день  → обсуждение
  ↓
 голосование → подтверждение казни
  ↓
 ...пока кто-то не победит
```

## Команды

| Команда | Где | Кто | Описание |
|--------|-----|-----|----------|
| `/start` | ЛС | все | Приветствие, язык, добавить в чат |
| `/lang` | ЛС | все | Язык личных сообщений |
| `/lang` | Группа | **владелец** | Язык объявлений в чате |
| `/game` | Группа | все | Создать набор |
| `/players` | Группа | все | Статус / список |
| `/skip` | Группа | хост | Досрочно завершить фазу |
| `/stop` | Группа | хост / админ | Остановить игру |

## Роли

| | Роль | Команда | Ночь |
|---|------|---------|------|
| 👤 | Мирный | Город | спит |
| 🤵 | Дон | Мафия | голос за жертву (решающий) |
| 😎 | Мафия | Мафия | голос за жертву |
| 🔍 | Комиссар | Город | проверка или выстрел |
| 👮 | Сержант | Город | видит проверки; наследует комиссара |
| 💊 | Доктор | Город | лечит (себя — 1 раз) |
| 🔪 | Маньяк | Нейтрал | убивает; победа в одиночку |
| 💋 | Любовница | — | блокирует цель на ночь + день |
| 💼 | Адвокат | — | клиент «мирный» при проверке |
| 💀 | Самоубийца | Нейтрал | победа при дневной казни |

| 💣 | Камикадзе | Город | при казни утягивает с собой |

Роли подключаются по числу игроков (от 4 до 20).

## Языки

Личный язык бота и язык чата **не зависят друг от друга**.

| Контекст | Меняет | Команда |
|----------|--------|---------|
| Личка | сам пользователь | `/lang` |
| Группа | только владелец чата | `/lang` |

Пример: в ЛС у вас русский, в чате владелец включил українську — роли приходят російською, а объявления в группе українською.

## Настройки `.env`

| Переменная | По умолчанию | Описание |
|------------|--------------|----------|
| `BOT_TOKEN` | — | токен от BotFather |
| `NIGHT_SECONDS` | `90` | длительность ночи |
| `DAY_SECONDS` | `180` | обсуждение |
| `VOTE_SECONDS` | `60` | голосование |
| `CONFIRM_SECONDS` | `30` | подтверждение казни |
| `MIN_PLAYERS` | `4` | минимум игроков |
| `MAX_PLAYERS` | `20` | максимум игроков |
